## TEST PLAN

LAYIHƏNİN ADI - Kapital Bankın mobil tətbiqi və veb sayti testingi.
Bu Test planı Kapital Bankın mobil tətbiqi və veb sayti üçün hazırlanıb.
Aparılacaq testlərin əsas məqsədi sistemin düzgün işlədiyini yoxlamaq, mümkün riskləri vaxtında aşkar edib müştəriyə keyfiyyətli məhsul təqdim etməkdir.
Tester: Sadıyeva Türkanə

## 1. TESTİN MƏQSƏDİ 

	Pul və hesabla bağlı edilən bütün əməliyyatlarının düzgün olmasını təsdiqləmək
	Müştəri məlumatlarının qorunmasını və təhlükəsizlyini təmin etmək
	Əsas və kritik əməliyyatlarda riskləri azaltmaq
	Müştəriyə stabil və etibarlı məhsul təqdim etmək

## 2. TEST SCOPE

	Login/Logout (OTP, PIN, biometrik)
	Balansın görünməsi
	Pul köçürmələri 
	Kommunal və digər ödənişlər
	Kredit və depozit əməliyyatları
	Bildirişlər (SMS, Push, Email)
	UI və UX yoxlamaları (usability)
	Sistemdəki dəyişikliklərin yenidən yoxlanması (Regression testləri)

## 3. TEST NÖVLƏRİ 

	Smoke Testing
	Functional Testing
	Integration Testing
	Regression Testing
	Usability Testing
	Compatibility Testing 
	Security Testing 
	Performance Testing 

## 4. TEST MÜHİTİ
